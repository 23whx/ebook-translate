## 🚀 5步快速获取

```
┌─────────────────────────────────────────────────────────┐
│  1️⃣  访问 https://huggingface.co/settings/tokens      │
│  2️⃣  点击 "New token" 按钮                             │
│  3️⃣  选择 "Read" 类型                                  │
│  4️⃣  输入名称（如：translategemma-api）                │
│  5️⃣  点击 "Create token"                               │
│  ⚠️  立即复制 Token！（只显示一次！）                   │
└─────────────────────────────────────────────────────────┘
```

---

## ⚙️ 配置选项

### Token 类型选择

| 选项 | 选择 | 原因 |
|------|------|------|
| **Read** | ✅ **选这个** | 安全够用 |
| Write | ❌ | 权限太大 |
| Fine-grained | ❌ | 太复杂 |

### Token 名称示例

```
✅ translategemma-api
✅ ebook-translator
✅ my-translation-app
✅ 任何您喜欢的名字
```

---

## 📋 Token 格式

```
✅ 正确格式：
hf_AbCdEfGhIjKlMnOpQrStUvWxYz1234567890

❌ 错误格式：
Token: hf_...     ← 不要包含 "Token:"
hf_AbC...         ← 不完整
```

## ⚡ 使用 Token

### 在我们的应用中

1. 打开 `http://localhost:3000`
2. 找到 "Hugging Face API Token" 输入框
3. 粘贴 Token
4. 点击 "验证"
5. 看到 "✓ Token 有效"

### 如果忘记保存

1. 访问 https://huggingface.co/settings/tokens
2. 删除旧 Token
3. 创建新 Token

## 🐛 故障排查

| 问题 | 解决方法 |
|------|----------|
| Token 无效 | 检查是否完整复制 |
| 验证失败 | 确认以 `hf_` 开头 |
| 忘记保存 | 重新创建一个 |
| 泄露了 | 立即删除并重新创建 |
